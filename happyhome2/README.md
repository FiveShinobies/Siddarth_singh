# Happy Home - Vite React Demo

This is a minimal Vite + React scaffold containing:
- About (Home) page
- ContactUs page
- Feedback component (form)

## How to run
1. unzip and `cd` into the project directory
2. run `npm install`
3. run `npm run dev`
4. Open http://localhost:5173

## Notes
- This is a frontend-only demo. Forms show demo alerts; connect to your backend for real submissions.
